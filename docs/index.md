##¿En qué consiste la actividad?

Los diferentes puntos de la actividad consisten en lo siguiente:

1. Añadir comentarios claros y correctos al código de la aplicación.
	- Demostrar que se entiende el código y se explica.
2. Ejecutar la aplicación desde el IDE y utilizar la depuración para analizar el comportamiento del programa.
	- Hacer un uso consciente del depurador.
3. Implementar los tests unitarios indicados y ejecutarlos correctamente.
	- Demostrar que el programa funciona correctamente.
4. Ejecutar la aplicación de modo seguro.
	- Utilizar un entorno aislado para ejecutar la aplicación y demostrar la importancia del aislamiento.

---

##Objetivos que pretendo alcanzar

- Demostrar que se comprende el funcionamiento del programa mediante comentarios claros, correctos y bien estructurados en el código fuente.
- Evidenciar el uso del IDE y del depurador para analizar la ejecución del programa, localizar errores y entender el flujo de la aplicación.
- Verificar el correcto funcionamiento del programa mediante pruebas unitarias que validen los distintos casos de uso y errores posibles.
- Ejecutar la aplicación en un entorno aislado para demostrar medidas de seguridad que reduzcan riesgos para el sistema.
- Analizar y reflexionar sobre las diferencias de seguridad entre lenguajes de programación y sus entornos de ejecución.
